## Como utilizar:

Na pasta raiz, crie um arquivo `.env` utilizando como base o arquivo `.env.example` e defina todas as variáveis de ambiente necessárias.


Logo após, instale as dependências e inicie o servidor:

```sh
$ yarn
$ yarn dev
```
